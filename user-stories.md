# User Stories

1. As a user, I want to register an account so that I can access the app.

2. As a user, I want to log into my account securely.

3. As a user, I want to see a home screen with meditation recommendations.

4. As a user, I want to view detailed information about each meditation.

5. As a user, I want to save my favorite meditations.

6. As a user, I want my favorites to be saved using local storage.

7. As a user, I want to access settings to customize the app.

8. As a user, I want to enable or disable notifications.

9. As a user, I want to fetch meditation data using an external API.
